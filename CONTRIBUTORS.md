## Recognition Guidelines

### How Contributors Are Added

Contributors are automatically recognized when they:

1. **Submit code changes** through pull requests
2. **Report issues** that lead to improvements
3. **Improve documentation** through any contribution
4. **Help other users** in discussions or issues
5. **Review pull requests** and provide constructive feedback
6. **Participate in project discussions** with valuable input

## Adding Yourself to This List

We welcome you to add yourself to this list! Here's how:

1. Fork this repository
2. Add your information to this file in the appropriate section
3. Submit a pull request
4. Include a brief description of your contribution

**Format:**

```markdown
**Your Name** ([@your-github-username](https://github.com/your-github-username))

- Brief description of your contributions
```

Alternatively, maintainers will add you when your contribution is merged.
